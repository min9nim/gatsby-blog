### 전체화면캡쳐
파일로 저장
```
Shift + command + 3
```

클립보드로 저장
```
Shift + command + Ctrl + 3
```
Note)
- 파일저장 경로는 데스크탑

<br>

### 선택윈도우 or 특정영역드래그
파일로 저장
```
Shift + command + 4
```

클립보드로 저장
```
Shift + command + Ctrl + 4
```

Note)
- 선택윈도우냐 특정영역드래그(default)냐는 스페이스키로 토글

<br>

### 동영상 캡쳐
```
Shift + command + 5
```

Note)
- 이거는 전체화면 + 선택윈도우 + 특정영역 + 동영상캡쳐 까지 종합선물세트로 기능 지원
- 기본적으로 파일로 저장
    - 클립보드로 저장하려면 Ctrl 누르고 "캡쳐" 클릭
- 동영상녹화 종료 단축키
    - Command + Ctrl + ESC
